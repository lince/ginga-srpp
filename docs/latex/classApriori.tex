\subsection{Detailed Description}
Essa classe implementa o algoritmo APRIORI.

O objetivo do algoritmo apriori é encontrar os conjuntos de dados frequentes L no banco de dados. Itens com pouca frequencia devem ser deletados permanecendo apenas aqueles que ocorrem mais vezes.
